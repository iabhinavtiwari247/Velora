## About Velora

**Velora** is India's most comprehensive matrimony-cum-dating platform, built for the modern Indian family. It blends the social experience of Instagram, the professional credibility of LinkedIn, and the community spirit of Facebook — all within a trusted, culturally rooted matrimony ecosystem.

Velora is designed for Indians aged **21–37 years**, with Aadhaar + PAN-based identity verification, physical home-visit trust verification by Field Agents, and completely free messaging — making it the only platform where finding a life partner feels as natural as scrolling your social feed.

---

| Principle | Velora's Approach |
|-----------|-------------------|

| **Transparency** | No fake profiles, no private accounts, no hidden charges for messaging |
| **Inclusivity** | Supports Boy, Girl, and Parent/Guardian registrations |
| **Cultural Sensitivity** | Caste, Gotra, Surname, Kundali Milaan — all built in |
| **Freedom** | Completely free core experience; premium subscriptions unlock lifestyle upgrades |

## Features Overview

### Registration Types

Velora supports three registration modes:

#### a) Register as Boy / Male
- Looking for a female partner
- Can only view female profiles

#### b) Register as Girl / Female
- Looking for a male partner
- Can only view male profiles

#### c) Register as Parent / Guardian
- Can browse both male and female profiles
- Can shortlist matches on behalf of their son or daughter

> **Age Gate:** Registration is restricted to individuals between **21 and 37 years of age**, verified against Aadhaar date of birth.

## Field Agent Programme

Velora creates **employment opportunities** for unemployed users (aged 21–37) by enrolling them as **Field Agents** — a trust-verification workforce similar to how Zomato delivery partners operate.

### How it Works

1. After two profiles mutually match, both families are offered an **optional physical verification visit**
2. A nearby Field Agent is assigned (like a delivery assignment)
3. The Field Agent visits the home of the user, verifies:
   - Physical presence and appearance matches profile photos
   - Family background is genuine
   - Address and employment claims are accurate
4. Agent submits a structured report via the Velora Agent App
5. Profile receives a **"Home Verified"** badge visible to all matches

### Agent Benefits
- Per-visit earnings (piecework model)
- Performance ratings and bonuses
- Weekly payout via UPI / bank transfer
- Field Agent badge on their own Velora profile
- Priority in their own matrimony match discovery

### Agent Requirements
- Velora-registered user aged 21–37
- Aadhaar + PAN verified
- Passed a background check
- Completed Velora Agent onboarding training (online, ~2 hours)

## Privacy & Safety

- **Gender-filtered browsing:** Boys see only girls' profiles; girls see only boys'. Parents can see both.
- **No private profiles:** All verified profiles are visible. Transparency builds trust.
- **No block feature:** Respectful interaction is encouraged; harassment is handled via Report + moderation.
- **Report & Flag system:** Any profile can be reported for fake information, inappropriate content, or misconduct
- **AI content moderation:** Reel and image uploads are scanned before publishing
- **Data encryption:** All messages are end-to-end encrypted
- **Aadhaar data:** Only age and name are stored post-verification; raw Aadhaar data is not retained
- **DPDP Act 2023 compliant** (India's Digital Personal Data Protection Act)
